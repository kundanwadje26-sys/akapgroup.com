Code Segments Explained

Partner Type (R/D/B/P)
Identifies whether the partner is a Retailer, Dealer, Builder, or Plumber in the distribution chain.
State Code (2 letters)
Standard Indian state abbreviation code (e.g. MH for Maharashtra, DL for Delhi).
City RTO Code
The RTO district code of the city where the partner operates.
Zone (U/R)
Urban (U) or Rural (R) — classifies the market type where the partner is located.
